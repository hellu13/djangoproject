# Django 프로젝트

## 프로젝트 소개
+ 간단한 블로그 형식의 게시판 구현

## 사용 언어 및 프레임 워크
+ 사용 언어 : HTML, CSS(BootStrap), JavaScript, Python
+ 사용 프로그램 : Visual Code
+ 사용 프레임워크 : Django
+ 사용 데이터베이스 : SQLite

## 구현 기능
+ 로그인/로그아웃
+ 글 작성/읽기/수정/삭제
